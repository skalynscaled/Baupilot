# BauPilot

Ein einfaches Next.js Projekt zur Generierung von Arbeitsplänen und Materiallisten für Bauunternehmen.

## Entwicklung

1. Installiere Abhängigkeiten
   ```bash
   npm install
   ```

2. Starte die Entwicklungsumgebung
   ```bash
   npm run dev
   ```

Die OpenAI-API wird mit einem Dummy-Key verwendet. Ersetze `DUMMY_KEY` in `pages/api/generate.ts` durch deinen Schlüssel.
